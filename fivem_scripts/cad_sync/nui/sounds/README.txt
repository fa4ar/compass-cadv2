CAD Sync Sound Files
====================

This directory should contain notification sound files.

Required files:
- alert.mp3 - Alert sound for 911 call notifications

You can use any MP3 file. For best results, use short (1-2 second) alert sounds.

Example sources for free sounds:
- Freesound.org
- Zapsplat.com
- Mixkit.co

Place your sound files in this directory and update the Config.Notifications.SoundFile setting in config.lua if using a different filename.
